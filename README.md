## 📌 Featured Projects

### 🏪 [goldshinetrade](https://github.com/tigranadamyan/goldshinetrade) — Gaming Currency Exchange Platform
Full-stack marketplace for exchanging in-game currencies. Features include escrow system, real-time chat (Pusher WebSockets), CKassa payment integration, VK ID + Google OAuth, admin panel, and Docker deployment.

`Laravel 12` `Vue 3` `Inertia.js` `Tailwind CSS` `Docker`

### 💻 [Configurator_PC](https://github.com/tigranadamyan/Configurator_PC) — PC Build Configurator
Interactive PC build configurator with component compatibility validation. Features a catalog with web-scraped pricing (RoachPHP spiders), build sharing with likes/comments, admin panel, and DeepSeek AI chat assistant.

`Laravel 12` `Blade` `Tailwind CSS` `Alpine.js` `MySQL`

### 🎮 [AOE2DEAnalyzator](https://github.com/tigranadamyan/AOE2DEAnalyzator) — AoE2 Match Analyzer
Analytics platform for Age of Empires II: Definitive Edition. Parses replay data, applies domain-driven analysis rules via a custom rules engine, and surfaces insights through a Vue 3 + Filament admin panel.

`Laravel 12` `Vue 3` `Filament` `Python` `Steam API`

### 📚 [Programm-Pedia](https://github.com/tigranadamyan/Programm-Pedia) — Developer Knowledge Base
Developer-focused knowledge base with practical code examples, error solutions, and step-by-step guides. Built with Meilisearch for instant full-text search, Laravel backend, and Vue 3 frontend.

`Laravel 12` `Vue 3` `Meilisearch` `Docker`

### 🤖 [titleSheet_bot](https://github.com/tigranadamyan/titleSheet_bot) — Isometric Sprite Sheet Generator
Telegram bot that resizes and removes backgrounds from images, optimized for isometric game sprites. Uses rembg (AI background removal) with 2-pass normalization pipeline in Docker.

`Python` `rembg` `Telegram Bot API` `Docker`

### 🖥️ [DropAndDrag](https://github.com/tigranadamyan/DropAndDrag) — Cross-Platform Drag & Drop Shelf
*Side project.* Native C++23 desktop utility for macOS and Linux. Shake cursor to open a floating shelf — drop files, images, text, URLs. Built with Skia rendering, Cocoa/X11 backends, and CMake.

`C++23` `Cocoa` `X11` `Skia` `CMake` `Homebrew`
